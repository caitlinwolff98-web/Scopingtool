## About This Archive

This folder contains **legacy documentation** from versions 2.0 through 4.0 of the Bidvest ISA 600 Scoping Tool. These documents have been archived because:

1. **Documentation Overload:** The repository had 30+ documentation files, creating confusion
2. **Redundancy:** Multiple files covered the same topics with slight variations
3. **v5.0 Consolidation:** All essential information has been consolidated into streamlined guides

## Migration Notes

If you're upgrading from v4.0 or earlier:

1. **VBA Code:** No changes required - v5.0 VBA is backward compatible
2. **Documentation:** Use new IMPLEMENTATION_GUIDE.md for setup
3. **Power BI:** Use new DAX_MEASURES_LIBRARY.md for comprehensive measures
4. **Verification:** Use new VERIFICATION_CHECKLIST.md for testing
